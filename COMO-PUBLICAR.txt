HUNTER — site estático pronto para publicar
============================================

Conteúdo desta pasta (tudo que o site precisa, sem build, sem internet em runtime):
  index.html, app.js, vendor/react*.js, hunter_logo_icon.png, .nojekyll

OPÇÃO 1 — Netlify Drop (mais rápido, ~30s, sem conta obrigatória)
  1. Abra https://app.netlify.com/drop
  2. Arraste ESTA pasta inteira para a área indicada.
  3. Pronto: você recebe uma URL pública na hora.

OPÇÃO 2 — GitHub Pages
  1. Crie um repositório vazio em https://github.com/new (ex.: "hunter").
  2. Na sua máquina:
       cd <esta-pasta>
       git init && git add -A && git commit -m "Hunter"
       git branch -M main
       git remote add origin https://github.com/<você>/hunter.git
       git push -u origin main
  3. No repo: Settings -> Pages -> Source: branch "main" / "(root)".
  4. Fica em: https://<você>.github.io/hunter/

OPÇÃO 3 — qualquer host estático (Vercel, Cloudflare Pages, S3, etc.)
  Suba o conteúdo desta pasta como raiz do site.
